ctrl + c :
	en input : cree une nouvelle ligne
	en here_doc : coupe le here doc
	en attente des childs : coupe les childs : verifier que ca coupe bien les ||, les && et les pipes
ctrl + d :
	en input : quitte le shell si la ligne est vide
	en here_doc : coupe le here_doc si la ligne est vide mais envoie quand meme l input
	en attente des childs : si le child attent de l input, coupe l input
ctrl + \
	en input : rien
	en here_doc : rien
	en attente des childs : coupe le child et core dump
